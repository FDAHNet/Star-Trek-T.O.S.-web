# Star Trek - Temporada 1

Sitio web en español dedicado a la primera temporada de **Star Trek: The Original Series**.

## Contenido

- Resumen general de la temporada
- Personajes principales y recurrentes de la tripulación
- Guía completa de episodios
- Buscador en tiempo real
- Ilustraciones SVG locales

## Desarrollo con npm

Instalación de dependencias:

```powershell
npm install
```

Servidor local de desarrollo:

```powershell
npm run dev
```

Compilar para producción:

```powershell
npm run build
```

Previsualizar la versión compilada:

```powershell
npm run preview
```

## Archivos principales

- `index.html`
- `styles.css`
- `game.js`
- `package.json`
